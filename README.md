# Laravel + Inertia + Tailwind + Mantine

```
$ git clone git@github.com:zushiK/laravel-inertia-template.git .
$ composer install
$ npm install
$ cp .env.example .env
$ php artisan key:generate
$ npm run dev
$ php artisan serve
```
